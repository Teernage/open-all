# 隐私政策 / Privacy Policy

**最后更新日期 / Last Updated:** 2025 年 11 月

## 中文版

### 概述

Open-All 是一个简单的浏览器扩展工具，旨在帮助用户批量打开多个 URL 链接。我们非常重视您的隐私，本隐私政策说明了我们如何处理您的信息。

### 我们收集什么数据？

**我们不收集任何用户数据。**

Open-All 扩展：

- ❌ 不收集个人身份信息
- ❌ 不收集浏览历史
- ❌ 不收集位置信息
- ❌ 不使用 Cookie 或追踪技术
- ❌ 不向任何第三方传输数据

### 本地存储

为了提升用户体验，Open-All 使用浏览器的本地存储功能（`chrome.storage.local`）来保存您在文本框中输入的 URL 链接。

**重要说明：**

- ✅ 所有数据仅存储在您的本地设备上
- ✅ 数据不会上传到任何服务器
- ✅ 数据不会与任何第三方共享
- ✅ 您可以随时通过清除浏览器数据来删除这些信息

### 权限说明

Open-All 请求以下权限：

1. **`tabs` 权限**

- 用途：在新窗口中创建标签页
- 说明：这是实现批量打开链接的核心功能所必需的

2. **`storage` 权限**

- 用途：在本地保存用户输入的链接内容
- 说明：仅用于恢复用户上次输入的内容，提升使用体验

### 数据安全

由于我们不收集或传输任何数据，因此不存在数据泄露的风险。所有操作都在您的本地设备上完成。

### 第三方服务

Open-All 不使用任何第三方服务、分析工具或广告网络。

### 儿童隐私

Open-All 不会故意收集 13 岁以下儿童的任何信息。由于我们不收集任何数据，因此适合所有年龄段的用户使用。

### 隐私政策变更

如果我们对隐私政策进行重大更改，我们会在扩展更新说明中通知用户，并更新本文档的"最后更新日期"。

### 联系我们

如果您对本隐私政策有任何疑问或建议，请通过以下方式联系我们：

- GitHub Issues: https://github.com/Teernage/open-all/issues
- GitHub Repository: https://github.com/Teernage/open-all

## English Version

### Overview

Open-All is a simple browser extension tool designed to help users batch open multiple URL links. We take your privacy seriously, and this privacy policy explains how we handle your information.

### What Data Do We Collect?

**We do not collect any user data.**

Open-All extension:

- ❌ Does not collect personal identification information
- ❌ Does not collect browsing history
- ❌ Does not collect location information
- ❌ Does not use cookies or tracking technologies
- ❌ Does not transmit data to any third parties

### Local Storage

To enhance user experience, Open-All uses the browser's local storage feature (`chrome.storage.local`) to save the URL links you enter in the text box.

**Important Notes:**

- ✅ All data is stored only on your local device
- ✅ Data is not uploaded to any servers
- ✅ Data is not shared with any third parties
- ✅ You can delete this information at any time by clearing your browser data

### Permission Explanation

Open-All requests the following permissions:

1. **`tabs` Permission**

- Purpose: To create tabs in a new window
- Explanation: This is essential for the core functionality of batch opening links

2. **`storage` Permission**

- Purpose: To locally save the link content entered by users
- Explanation: Only used to restore the user's last input, improving user experience

### Data Security

Since we do not collect or transmit any data, there is no risk of data breaches. All operations are completed on your local device.

### Third-Party Services

Open-All does not use any third-party services, analytics tools, or advertising networks.

### Children's Privacy

Open-All does not knowingly collect any information from children under 13. Since we do not collect any data, it is suitable for users of all ages.

### Changes to Privacy Policy

If we make significant changes to our privacy policy, we will notify users in the extension update notes and update the "Last Updated" date in this document.

### Contact Us

If you have any questions or suggestions about this privacy policy, please contact us through:

- GitHub Issues: https://github.com/Teernage/open-all/issues
- GitHub Repository: https://github.com/Teernage/open-all
